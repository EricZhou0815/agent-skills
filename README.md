# Antigravity Agent Skills

A collection of specialized AI skills designed to superpower your job search and engineering workflows.

## 🛠️ Skills Overview

### 1. Skill Generator
**What it does**: Creates new Antigravity skills from scratch in minutes.
**How to use**:
- Ask: *"Help me create a skill for [Topic]"*
- Features: Quick-start workflow, automation scripts, and ready-to-use templates for instruction-only or script-based skills.

### 2. NZ Job Search
**What it does**: Finds current NZ software engineering leadership jobs with strict filtering (Location, Recency, Leadership Scope).
**How to use**:
- Ask: *"Find NZ tech lead jobs posted this week"*
- Features: Filters for Christchurch/Remote-NZ, verifies posting dates (last 7 days), and excludes unrelated roles.

### 3. Application Coach
**What it does**: Evaluates your CV against a JD from a **Hiring Manager's perspective**.
**How to use**:
- Ask: *"Review my CV for this job"*
- Features: Strict risk assessment, identifying "Gatekeepers" vs "Differentiators", and providing a "Hire/No Hire" probability simulation.

### 4. Cover Letter Writer
**What it does**: Turns standard cover letters into **strategic risk-mitigation documents**.
**How to use**:
- Ask: *"Write a cover letter for this application"*
- Features: Uses the **Hook-Bridge-Close** framework to address "The Elephant in the Room" (e.g., missing skills) and align with company mission.

## 📂 Directory Structure

- `.agent/skills/`: Contains the source code for all skills.
- `input/`: Place your private CVs and JDs here (ignored by git).
- `output/`: Generated reports and drafts will be saved here.
